Pseudo Classes


nth-of-type

  • n can be a number, a keyword (odd or even)
  • This selector matches every element that is the nth child, of the same type (tag name), of its parent.
  • e.g :nth-last-of-type(2) Selects only 2nd last element of its parent

not Selector

This selector matches every element that is NOT the specified element/selector.

Here it will set a background color for all elements that are not a paragraph

lang

Selector is used to select elements with a lang attribute with the specified value.
